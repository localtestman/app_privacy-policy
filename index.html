👋 හඳුන්වාදීම / Introduction

NetWedha (com.sripci.netvistasl) යනු Sri Lanka හි network monitoring, signal analysis, speed testing සහ community features සඳහා නිර්මාණය කරන ලද Android application එකකි.

NetWedha is an Android application designed for Sri Lanka, providing network monitoring, signal analysis, speed testing, and community features via NetSabawa.

📌 Important මෙම privacy policy එක app version 2.03.1 සඳහා අදාළ වේ. App download හෝ install කිරීමෙන් ඔබ මෙම policy එකට එකඟ වෙයි.

📊 එකතු කරන දත්ත / Data We Collect

App එක භාවිතා කිරීමේදී පහත දත්ත එකතු කළ හැකිය:

  • Location Data: Network signal strength map කිරීම සහ GPS test features සඳහා ඔබේ location භාවිතා කෙරේ.
  • Network Information: WiFi SSID, signal strength, connection type, IP details.
  • Device Information: Phone model, OS version, SIM carrier details, phone number (Dual SIM Comparator සඳහා).
  • Usage Data: App features භාවිතා කරන ආකාරය, error logs.
  • NetSabawa Content: ඔබ post කරන content, comments, images/videos.
  • Speed Test Results: Download/upload speed, latency measurements.
⚠️ Note Location data real-time වශයෙන් server එකට නොයවනු ලැබේ. Signal mapping features සඳහා පමණක් භාවිතා කෙරේ.
